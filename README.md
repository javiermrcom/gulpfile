# gulpfile

You can use this *gulpfile* to your projects.

### Installation
```
npm install
```

### Configuration
You only have to change the value of each path.
``` javascript
var path = {
  dist: './public',
  src: './resources'
};
```

### Use
#### Development task
Run in command line:
```
gulp dev 
```
#### Production task
Run in command line:
```
gulp dist 
```
